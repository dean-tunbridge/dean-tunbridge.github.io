What is JavaScript, and how does it relate to HTML and CSS

In my reflections blog I made the analogy of HTML and CSS being like a house, so I will try and make this work when adding JavaScript! I compared HTML to the frame/foundations of a house, and CSS being the plaster, fittings and paint. Carrying this on, I would say JavaScript could be compared to the electrical wiring, plumbing, and doors! I hope this analogy makes sense!

Imagine the framework of construction like walls, floors, and rooms in a house which would translate to being elements like the body, headings, divs, paragraphs, images etc in our HTML. It is essential to make this part first, as just like in a house, You can't paint the walls before the walls have been built!

CSS is time to bring in the interior decorators, designers and painters. This is to make the house/Webpage visually appealing. It decides the colors, fonts, spacing, and overall look of each of the HTML elements. You could paint the whole house one colour, but maybe that would be boring, so you can define each element with a class, which means you can design, layout and paint each room/element individually.

So you have built your house, it's now decorated perfectly how you wanted it, but none of the lights, taps, or doors are working! This is where JavaScript comes in. If you imagine you have your light fittings, and light switches installed, JavaScript is the wiring that tells the switch what to do 'onclick'. In the same way that when you turn on the shower, you expect water to come out. Without the plumbing/JavaScript this wouldn't be possible.

Control flows

Control flow is the order in which JavaScript executes statements. An example would be an if else statement, Above is an if else statement I made to demonstrate this. The if statement evaluates a condition inside parentheses (x > 5), If this condition is true, the code following the if statement (console.log('x is greater than 5'))executes. If the condition is false, the code inside the else statement (console.log('x is not greater than 5')) executes.
